Установка

Вариант 1 - Перетащить файл MTL_2.4.mzp во вьюпорт 3dsmax.
Вариант 2 - Добавление через меню Scripting -> Run Script -> MTL_2.4.mzp

Перед применением рекомендуется перезагрузить 3dsmax.

Добавление кнопки на тулбар:
Customize -> Customize User Interface -> Toolbar -> Category:Val -> Mat_Tex_Loader

Начало работы

Скрипт предназначен для сборки Physical или VrayMat материалов
из набора текстур, подготовленных в соответствии с ТЗ.
Материалы, создаваемые скриптом, в большинстве случаев требуют донастройки

Physical - создание материала Physical Material (по-умолчанию)
Vray - создание материала VrayMat (необходимо согласовывать с менеджером проекта)
Load Textures - выбор набора текстур относящихся к создаваемому материалу
Create - подтверждение, что текстры выбраны верно и создание материала
Mateditor Slot - вывод материала в Material Editor
Clear - очистка выбранных текстур (если выбраны неправильно)
Set reflection - установка значения reflection материала
Set refl IOR - установка значения reflection IOR материала (по умолчанию 1.5)
